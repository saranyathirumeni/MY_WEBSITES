HTML5

CSS

JavaScript

A programming language of HTML and the Web. Makes computers do what you want them to do.

JQuery

A cross-platform JavaScript library designed to simplify the client-side scripting of HTML.

A set of web development techniques using many web technologies on the client-side to create asynchronous Web applications.
